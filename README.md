# Interactive Backgrounds

Коллекция интерактивных задних планов для веб-сайтов.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Проект откроется в браузере на `http://localhost:9000`

## Сборка

```bash
npm run build
```

## Интерактивные фоны

1. **Частицы** - Частицы, которые следуют за курсором и соединяются линиями
2. **Волны** - Анимированные волны, реагирующие на движение мыши
3. **Геометрия** - Геометрические фигуры, появляющиеся при клике
4. **Градиент** - Плавный цветной градиент, следующий за курсором
5. **Параллакс** - Сетка с эффектом параллакса при движении мыши
